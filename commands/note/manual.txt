Write note for the server

Required permissions:
    !banned
    
```
Usage:
    note <name> <value>       : Write note for server
    note -u <name> <value>    : Write note only for current user
    note -o <name> <value>    : Overwrite note for server
    note -o -u <name> <value> : Overwrite note only for current user
    note -d <name>            : Delete note
    note -d -u <name>         : Delete note only for current user
    note -l                   : List all notes for server
    note -l -u                : List all notes for current user
```
