Install package to the system

```
Options:
    --ignore-dependencies: [Install / Remove] Ignore dependencies
    --ignore-conflicts:    [Install] Ignore conflicts
    --chain:               [Remove] Remove all packages that depend on the package
    --reinstall:           [Install] Reinstall the package even if it is already installed
```
```
Usage:
    packtool install <url...> [options]  : Install packages
    packtool remove <id...> [options]    : Remove packages
    packtool list                        : List installed packages
```
